# 🛂 نظام قراءة وفحص الهويات وجوازات السفر البيومترية عبر NFC

نظام متكامل مفتوح المصدر لقراءة البيانات البيومترية المشفرة (مثل الاسم الكامل، تاريخ الميلاد، وصورة الوجه الشخصية) من جوازات السفر وبطاقات الهوية المتوافقة مع معيار **ICAO 9303** العالمي، ونقلها لحظياً وعرضها على الكمبيوتر.

---

## 🛠️ بنية النظام (System Architecture)

يتكون النظام من جزأين رئيسيين يتواصلان عبر الشبكة المحلية (WiFi):

1. **تطبيق الهاتف (Mobile Client - React Native / Expo)**:
   * يقوم بالاتصال المباشر بالبطاقة الذكية باستخدام بروتوكول **IsoDep** و الـ NFC.
   * يقوم بعمل مصادقة **BAC (Basic Access Control)** لفك تشفير بيانات البطاقة باستخدام حقول الـ MRZ.
   * ينشئ قناة اتصال آمنة ومُشفرة (**Secure Messaging**) لقراءة ملفات البيانات الثنائية:
     * `EF.DG1`: بيانات الهوية النصية (MRZ).
     * `EF.DG2`: الصورة البيومترية الشخصية للوجه.
   * يرسل البيانات ثنائية مشفرة بصيغة Base64 إلى خادم الكمبيوتر.

2. **برنامج الكمبيوتر (Desktop Server - FastAPI / Web UI)**:
   * خادم محلي خفيف وسريع مبني باستخدام بايثون (`FastAPI`).
   * يقوم بتحليل ملفات الـ ICAO وتوليد كافة بيانات الوثيقة.
   * يفك ضغط الصورة الشخصية (والتي تكون بصيغة **JPEG2000** الثنائية) ويحولها لصيغة PNG قابلة للعرض.
   * يبث البيانات لحظياً عبر **WebSockets** إلى واجهة مستخدم ويب فائقة الجمال (Dark Glassmorphic) مع إصدار تنبيهات صوتية عند القراءة.

## 🚀 التشغيل السريع (Quick Start)

### أولاً: تشغيل برنامج الكمبيوتر (Server)

1. يجب أن يكون لديك [Python 3.8+](https://www.python.org/) مثبتاً.
2. افتح موجه الأوامر في مجلد المشروع وانتقل إلى مجلد الخادم:
   ```bash
   cd desktop-app
   ```
3. ثبّت الحزم البرمجية المطلوبة:
   ```bash
   pip install -r requirements.txt
   ```
4. ابدأ تشغيل الخادم:
   ```bash
   python main.py
   ```
5. افتح المتصفح وتوجه إلى: `http://localhost:8000`.

---

### ثانياً: تثبيت تطبيق الهاتف (Mobile Client)

لقد تم إعداد **GitHub Actions** ليقوم ببناء ملف التثبيت تلقائياً سحابياً.
1. اذهب لتبويب **Actions** في مستودعك على GitHub.
2. قم بتحميل أحدث ملف APK منتج من قسم **Artifacts** باسم `nfc-sender-debug-apk`.
3. قم بتثبيته وتأكد من اتصال الهاتف بالكمبيوتر على **نفس شبكة الـ WiFi**.
4. أدخل عنوان الـ IP الظاهر على شاشة خادم بايثون بالكمبيوتر داخل التطبيق، واضغط **اختبار الربط**.

---

## 🛃 كيفية قراءة المستندات البيومترية

1. افتح التطبيق في الهاتف واختر **وضع الـ MRZ**.
2. أدخل المعلومات المطلوبة (رقم الوثيقة/الجواز، تاريخ ميلادك، وتاريخ انتهاء الوثيقة).
3. اضغط على **ابدأ مسح الوثيقة** وقرب الكارت أو الجواز من شريحة الـ NFC بالهاتف حتى تكتمل القراءة 100%.
4. ستظهر كافة المعلومات والصورة فوراً على شاشة الكمبيوتر!

*(يدعم التطبيق أيضاً القراءة التلقائية للمعرف الفريد UID لأي كروت ذكية عادية وغير بيومترية كبديل تلقائي).*
